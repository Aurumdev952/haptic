# **App Name**: Haptic

## Core Features:

- Speech Transcription: Continuously capture and transcribe speech from the microphone in Kiyarwanda and Kiswahili. It waits for a pause to finalize the transcription, handles noise effectively
- Text Display: Display the transcribed text in a clean and readable format.
- AI Response Generation: Use an LLM to generate three contextual response options to the transcribed input; it will use its discretion as a tool for generating them.
- Response Selection UI: Display the AI-generated response options as big, easy-to-tap buttons.
- Spoken Output: Speak aloud the selected response using a text-to-speech engine.
- Automatic Reset: Reset response options when new speech input is detected. and then add the new detected input to the converstation then generate new options

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and reliability.
- Background color: Light blue (#E8EAF6), a desaturated variant of the primary, for a calm feel.
- Accent color: Violet (#7E57C2), to highlight selectable elements and actions. Provides enough contrast with the primary for good affordance.
- Clean and modern sans-serif font for readability.
- Simple one-page layout with clear sections for input and output.
- Minimalist icons to represent speech, language, and interaction elements.
- Subtle animations to indicate processing and feedback.
